# Stock CrewAI - 虚拟盘自动交易系统

## 功能完成 ✅

### 1. 核心功能
- ✅ 新浪实时行情（稳定、快速、无限制）
- ✅ 自动买入（技术面筛选 + 风险管理）
- ✅ 自动卖出（止损8% / 止盈20% 触发）
- ✅ 持仓跟踪（盈亏实时计算）

### 4. 策略说明

#### 买入条件
- 今日温和上涨（0% ~ 8%）
- 有正常成交量
- 股价 ≤ 200元（确保单只≤2万）
- 已有持仓跳过

#### 仓位管理
- 单只最大：总资金20%（2万元）
- 牛市总仓位：≤80%
- 熊市总仓位：≤30%

#### 止损止盈
- 止损：-8% 自动卖出
- 止盈：+20% 自动卖出
- 风险收益比：≥1:2
